Character

What is a character in programming?

A character is any letter, number, space, punctuation mark, or symbol that can be typed on a computer.
The word "computer," for example, consists of eight characters.


What is ASCII code?

ASCII is the acronym for the American Standard Code for Information Interchange.
It is a code for representing 128 English characters as numbers, with each letter assigned a number from 0 to 127.
For example, the ASCII code for uppercase M is 77.
