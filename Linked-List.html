What is the Linked-List in C?

A linked list is a sequence of data structures, which are connected together via links.
Linked List is a sequence of links which contains items. Each link contains a connection to another link.
Linked list is the second most-used data structure after array.

Pointer

A pointer is a variable that stores the address of another variable.
Unlike other variables that hold values of a certain type, pointer holds the address of a variable.
Linked-Lists connect each other by using pointer. Each link contains a pointer which holds the other linked-list's address.
